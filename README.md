# Система управления библиотекой книг

Java-приложение для управления библиотекой с использованием продвинутых возможностей Git.

## Структура проекта
- `Book.java` - класс книги
- `Library.java` - класс библиотеки со вложенным классом журнала операций
- `Main.java` - демонстрационный класс

## Возможности
- Добавление/удаление книг
- Поиск по ID и автору
- Выдача и возврат книг
- Журналирование операций
- Статистика библиотеки

## Git команды, использованные в проекте
- `rebase` - перебазирование веток
- `reset` - отмена коммитов
- `cherry-pick` - выборочное применение коммитов
- `merge` - слияние веток
